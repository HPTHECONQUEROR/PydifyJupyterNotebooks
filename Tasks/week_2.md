Check Divisibility by Both 3 and 5:
Input a number. Print “FizzBuzz” if divisible by both 3 and 5, “Fizz” if only 3, “Buzz” if only 5, else the number itself.

Leap Year Check:
Input a year and print whether it’s a leap year or not.

Vowel or Consonant:
Input a single alphabet and print if it’s a vowel or consonant.

Count Even and Odd Digits:
Input a number. Count how many even and odd digits are present in it.

Power of a Number:
Input a base and exponent. Find the result using a loop (no **).

Sum of Multiples of 3 or 5 below N:
Input N. Find the sum of all numbers below N that are multiples of 3 or 5.

Palindrome Number:
Input a number. Check if it reads the same forward and backward.

Armstrong Number:
Input a number. Check if it’s an Armstrong number (sum of cubes of digits = number).
